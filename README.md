#Real Time
>a real-time voting app that uses client side security, WebSockets and jQuery.

##Built With:
* Node.js
* Express.js
* JavaScript
* jQuery
* Lodash
* Auth0
* WebSockets
* HTML/CSS

##View
[Real Time](https://real-time-laceyk.herokuapp.com/)

##Installation and Setup:
####Install
'npm install'

####Start
'npm start'
'visit Localhost://3000

####Test
'npm test'
